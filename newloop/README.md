# NewLoop Streamlit App

NewLoop is a simulation app built with Streamlit.

## Run locally

```bash
pip install -r requirements.txt
streamlit run slnewloop.py
```

## Deploy on Streamlit Community Cloud

1. Push this folder to a public GitHub repository.
2. In Streamlit Community Cloud, create a new app from that repo.
3. Set the main file path to `slnewloop.py`.

## Files

- `slnewloop.py`: Streamlit entrypoint
- `requirements.txt`: Python dependencies
- Core model modules: `engine.py`, `results.py`, and related files
